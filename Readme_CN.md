# 蛇咬病院坂saki音效

[English](Readme.md) | [中文](Readme_CN.md)

## 描述

杀戮尖塔2的mod. 在使用"蛇咬"这张牌时播放病院坂saki的嗷呜.

## 安装(Windows)

从Release下载最新的压缩包, 解压后将整个文件夹放入游戏根目录下的`mods`文件夹中即可.

``` text
root 
├── mods
│   ├── SnakeBiteBingyuanbanSakiSfx
│   │   ├── SnakeBiteBingyuanbanSakiSfx.dll
│   │   ├── SnakeBiteBingyuanbanSakiSfx.json
│   │   └── SnakeBiteBingyuanbanSakiSfx.pkg
```

如果没有`mods`文件夹, 请自己创建一个.

如果第一次安装mod, 请注意mod存档与原游戏存档独立. 在安装完mod后启动游戏时选择允许加载mod, 并到`%AppData%\SlayTheSpire2\steam\一串数字`目录下将`profilex`文件夹复制`modded`文件夹中, 即可复制原游戏存档到mod存档中.
